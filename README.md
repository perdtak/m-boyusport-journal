# m-boyusport-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容彼此独立，不依赖于任何特定域名或网站环境，可作为静态资源直接访问。

## 项目简介

`m-boyusport-journal` 是一个面向个人或小型团队的 HTML 页面归档仓库。主要用途包括：

- 保存和发布单页 HTML 文档
- 记录特定时间节点生成的页面快照
- 存放不依赖后端服务的纯前端页面

每个 HTML 文件均为自包含的静态页面，可离线浏览或部署至任意静态托管服务。

## 目录说明

仓库根目录下存放各 HTML 文件，文件命名遵循以下约定：

- 使用英文或拼音命名，避免特殊字符
- 文件名反映页面内容或生成日期

示例目录结构：

```
m-boyusport-journal/
├── index.html
├── 20250101-example.html
├── notes.html
└── ...
```

若未来文件数量增多，可能会按功能或时间创建子目录，届时会同步更新本说明。

## 页面归档说明

每个 HTML 页面均为完整、独立的文档，包含以下特性：

- **自包含**：所有样式、脚本和资源均内嵌或通过相对路径引用，不依赖外部 CDN 或特定网络环境
- **无外部依赖**：页面内容不涉及推广链接、跟踪代码或第三方服务
- **可离线访问**：保存至本地后可直接打开使用，无需联网

归档页面可能包含个人笔记、临时工具页面、实验性前端演示或其他非商业性内容。

## 维护说明

- 本仓库不定期更新，新增或修改页面时会通过 Git 记录变更
- 不保证所有页面的长期兼容性或可用性
- 欢迎通过 Issues 提交反馈，但不承诺处理周期
- 若需引用仓库中的页面，请保留原始文件完整性

## 使用建议

- 可通过 `git clone` 或直接下载 ZIP 获取全部文件
- 将仓库部署至 GitHub Pages 或其他静态托管服务后，即可在线访问各页面
- 如需修改内容，请 Fork 后自行维护
